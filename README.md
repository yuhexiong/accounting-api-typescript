# Accounting Server

## Overview

- Language: Typescript
- Web FrameWork: Express
- DataBase: MariaDB v10.5

## ENV

copy .env.example and rename as .env

```bash
DB_HOST=
DB_PORT=
DB_USER=
DB_PASS=
DB_NAME=

PORT =
```

## Run

```bash
# install dependencies
npm install

# run
npm start

```
